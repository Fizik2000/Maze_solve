# Maze_solve
Finding the shortest exit from a maze using Gauss's algorithm
  
## Технологии
**Стек технологий**  
Язык программирования C/С++  
Makefile  
CMake  
Latex  
Тестирование  
  
**Инструменты**  
Git, GitHub, Gtest, gcov, lcov  
  
## Установка и запуск  
Загрузка приложения - make install  
Открытие приложения - make start  
Тестирование - make test  
Проверка на покрытие кода - make gcov_report  
Проверка на стиль кода google - make clang_format  
Приведение к стилю кода google - make format  
Удаление объяктных файлов - make clean
